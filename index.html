Introduction

AICA-Bench is a benchmark framework for evaluating vision-language models on affective image content analysis. The repository organizes dataset construction, instruction generation, model adaptation, and benchmark evaluation into one reproducible codebase.

The current public release is repository-first: benchmark YAMLs, CLI entrypoints, prompt builders, model interfaces, preprocessing examples, and tests are already available. This webpage now follows the structure of an academic benchmark homepage so the final paper assets can be inserted directly later.

Overview

AICA-Bench targets multimodal affective evaluation with three major task families: emotion understanding, emotion reasoning, and emotion-guided generation. The repository also includes reusable components for building benchmark subsets and prompt variants across multiple datasets.
