Benchmark vision-language models through the lens of emotion.

AICA-Bench provides a unified evaluation pipeline for emotion understanding, emotion reasoning, and emotion-guided content generation. It packages dataset building, instruction construction, model adaptation, and metric computation into one reproducible workflow.

Overview

One benchmark, from dataset construction to final scoring.

Dataset building

Generate benchmark subsets with random or balanced sampling from emotion-focused datasets.

Instruction building

Produce standard prompts or CoT-style instructions for different evaluation settings.

Model adaptation

Run open-source or closed-model benchmarks through a consistent CLI interface.

Metric computation

Evaluate classification metrics and LLM-based metrics under a unified benchmark workflow.

Dataset access note

Workflow

Three commands to move from raw datasets to benchmark runs.

Step 1

Build benchmark subsets

aica-vlm build-dataset run benchmark/example.yaml --mode random
Step 2

Generate instructions

aica-vlm build-instruction run examples/config-yaml/instructions-builder/EU_observer_emotion_emoset.yaml
aica-vlm build-instruction run-cot examples/config-yaml/instructions-builder/EU_observer_emotion_emoset_CoT.yaml
Step 3

Run the benchmark

aica-vlm benchmark benchmark/example.yaml
