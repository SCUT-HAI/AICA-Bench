Introduction

AICA-Bench is a benchmark framework for evaluating vision-language models on affective image content analysis. The repository exposes a command-line workflow for building benchmark datasets, generating instructions, running model adaptation pipelines, and computing benchmark metrics.

The current release focuses on practical benchmark operations rather than a paper-style showcase. This page therefore highlights the assets already included in the repository: benchmark YAMLs, instruction builders, model interface modules, preprocessing examples, and test files.

Benchmark Tasks

Emotion Understanding

Evaluate whether models can identify and interpret emotional cues from visual inputs and benchmark datasets.

Emotion Reasoning

Measure how well models explain affective evidence and reason about the emotional content behind an image.

Emotion-Guided Generation

Assess whether models can produce outputs that follow emotion-oriented instructions and target styles.

Datasets and preprocessing examples

Included examples cover AffectNet, EmoSet, Emotic, Emotion6, FI, FlickrLDL, TwitterLDL, ArtPhoto, Abstract, FindingEmo, and related preprocessing scripts in examples/data_preprocessing.

Model adaptation templates

The repository already contains open-source model templates for InternVL, Qwen2-VL, Qwen2.5-VL, MiniCPM, Ovis2, and LLaVA across generation, reasoning, CoT, and zero-shot settings.

Repository Resources

CLI workflow

  • aica-vlm build-dataset run ... for benchmark subset construction
  • aica-vlm build-instruction run ... for instruction generation
  • aica-vlm benchmark ... for benchmark execution
  • aica-vlm compute-llm-metrics ... for LLM-based metrics

Project structure

  • src/aica_vlm contains the benchmark package
  • benchmark/ includes a starter benchmark YAML
  • examples/ contains adaptation and preprocessing examples
  • test/ provides repository test coverage

Dataset Access Note

This repository includes benchmark code, example configuration files, and processing utilities. Some underlying image datasets are derived from third-party emotion datasets and may require separate access requests or compliance with their original licenses.

Quick Start

Installation

pip install -e .
pip install -r requirements.txt

Example commands

aica-vlm build-dataset run benchmark/example.yaml --mode random
aica-vlm build-instruction run examples/config-yaml/instructions-builder/EU_observer_emotion_emoset.yaml
aica-vlm benchmark benchmark/example.yaml
