Introduction

AICA-Bench is organized as a reusable affective VLM benchmark

AICA-Bench is designed to evaluate vision-language models on affective image content analysis. The repository already unifies dataset construction, instruction generation, model adaptation, and benchmark evaluation into one public codebase.

The goal of this project page is to present that benchmark in the form of a research homepage rather than a repository landing page. The current public release focuses on real benchmark assets already available in the repository while reserving space for paper figures, leaderboard tables, and citation metadata.

Repository structure

  • src/aica_vlm contains the benchmark package.
  • adaptation/ includes model interfaces and runners.
  • instructions/ provides prompt-building modules.
  • metrics/ contains evaluation implementations.

Usage workflow

aica-vlm build-dataset run benchmark/example.yaml --mode random
aica-vlm build-instruction run examples/config-yaml/instructions-builder/EU_observer_emotion_emoset.yaml
aica-vlm benchmark benchmark/example.yaml
